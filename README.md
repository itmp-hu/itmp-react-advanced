# Haladó szintű React fejlesztés
## ITMP Klub képzés infotanároknak

A tavalyi "Bevezetés a Reac.js-be" tréningünk követően az ITMP Klub következő izgalmas képzése a React fejlesztés következő szintjére vezet benneteket. November 21-én, online formában egy haladó szintű workshopon vehetnek részt klubtagjaink, ahol egy valósághű projektet építünk fel lépésről lépésre.  
Szeretnéd, hogy Te és a tanítványaid ne csak az alapokon ragadjatok meg, hanem komplex React alkalmazások fejlesztésében is magabiztosak legyenek? Ez a képzés pont erre ad lehetőséget!  
A nap során a MITS (Marketable IT Skills) projektadatbázis egyik feladatát dolgozzuk fel, ahol megmutatjuk, hogyan készíts többoldalas, REST API-val kommunikáló webalkalmazást. Látni fogod, hogyan épül fel egy professzionális React projekt, hogyan kezeljük az állapotokat, és hogyan integráljuk a külső szolgáltatásokat.  
Az oktatást vezető mentorként Mrakovics Olivér Európa- és világbajnok webfejlesztőnk, valamint felkészítője, Sisák Zoltán vezetiks. Csatlakozz hozzánk, és szerezd meg azokat a haladó ismereteket, amelyekkel következő óráidon már komplex React projekteket vihetsz tanítványaiddal!

**FONTOS:** A képzésre csak azokat várjuk, akik már rendelkeznek a "Bevezetés a React.js-be" képzésünknek megfelelő szintű React alapokkal. Az önállóan is elvégezhető korábbi képzésünk anyagai [elérhetők az ITMP Klub szolgáltatásai](https://itmp.hu/klub/kartya/137)  között
